# project-docs

个人技术笔记与少量示例代码，主题覆盖 **Go**、**云原生（Kubernetes / Docker）**、**分布式系统**、**AI Agent / RAG / MCP**、**消息与数据（Kafka、Redis、向量检索）** 等。

## 内容说明

- 根目录下的大量 `.md` 为独立笔记，可按文件名在仓库内搜索浏览。
- **[kafka-analytics](./kafka-analytics/)**：Kafka 生产/消费相关的 Go 示例（见该目录下的 `README.md`）。

## 使用方式

克隆后直接打开关心的 `.md` 文件即可；若需运行 `kafka-analytics`，请进入子目录并参阅其中的说明。
