**多模态大模型（Multimodal Large Models）**指的是能够**同时理解和生成多种数据模态（modalities）**的人工智能模型，而不仅仅是文本。

简单理解：

> 传统大模型 = 只懂文字
> 多模态大模型 = 能同时理解 **文字 + 图片 + 视频 + 音频 + 甚至3D**

---

## 一、什么是“模态”（Modalities）

模态就是信息的表现形式，例如：

| 模态       | 示例             |
| ---------- | ---------------- |
| 文本       | 文章、聊天、代码 |
| 图像       | 照片、截图       |
| 视频       | 抖音视频、电影   |
| 音频       | 语音、音乐       |
| 结构化数据 | 表格、传感器数据 |
| 3D / 空间  | 自动驾驶、机器人 |

多模态模型可以把这些信息**统一到一个模型里理解**。

---

## 二、多模态大模型的核心能力

### 1 视觉理解（Vision）

例如：

输入图片 → 输出描述

例子：

图片：一只狗在沙滩上跑
模型输出：

> “一只金毛犬正在沙滩上奔跑，背景是海浪。”

典型应用：

- 图片识别
- 图像描述
- OCR
- 医学影像分析

---

### 2 图文理解（Vision + Language）

例如：

用户问：

> 这张图里的车是什么型号？

模型：

- 识别图片
- 理解问题
- 回答

典型任务：

- **VQA (Visual Question Answering)**
- 文档理解
- UI理解

---

### 3 文生图（Text → Image）

输入：

> “一只宇航员猫在月球喝咖啡”

模型生成图片

代表模型：

- DALL·E
- Stable Diffusion
- Midjourney

---

### 4 图生文（Image → Text）

输入：图片

输出：描述、故事、标签

应用：

- 自动生成 alt text
- 内容审核
- 商品描述生成

---

### 5 视频理解

输入：视频

模型可以：

- 总结视频
- 识别动作
- 转写字幕

例如：

抖音视频 → 自动生成脚本

---

### 6 语音能力（Speech）

输入语音：

> “帮我订一张机票”

模型：

1. 语音识别
2. 理解意图
3. 执行任务

典型模型：

- Whisper

## 四、多模态模型核心技术

主要有三类架构：

### 1 Encoder + LLM

结构：

```
Image → Vision Encoder → LLM → Text
```

例：

```
Image → CLIP → GPT
```

视觉模型：

- CLIP
- ViT

---

### 2 Unified Transformer

所有模态统一 token

例如：

```
image token
text token
audio token
```

放进一个 Transformer

代表：

- Gemini
- GPT-4o

---

### 3 Diffusion 模型（生成类）

用于：

- 文生图
- 文生视频

例如：

- Stable Diffusion

---

## 六、多模态模型技术难点

### 1 模态对齐（Alignment）

需要让模型理解：

```
图片 <-> 文字
```

例如：

狗 ↔ dog

---

### 2 数据成本

需要：

```
image + caption
video + subtitle
```

数据非常贵。

---

### 3 计算成本

视频模型计算量巨大。
